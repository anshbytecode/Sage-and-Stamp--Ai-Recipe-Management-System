# 🍽️Sage & Stamp — AI Recipe Management System

Sage & Stamp is an AI-powered recipe management and recommendation system that helps users add, organize, and discover recipes based on ingredients, cuisine type, and user preferences.

It uses **Machine Learning (TF-IDF, Naive Bayes, and K-Means clustering)** to classify recipes, predict cuisines, and recommend similar dishes — all running locally without external APIs.

---

## 🚀 Features

- ➕ Add, edit, and delete recipes
- 🔍 Search recipes by ingredients or name
- 🧠 AI-based cuisine classification (Naive Bayes)
- 🤖 Recipe recommendation system (TF-IDF similarity)
- 📊 Recipe clustering using K-Means
- 🥗 Tag-based filtering (veg, non-veg, quick, etc.)
- 🌐 Flask web interface
- ⚡ Fully offline ML processing (no APIs required)
